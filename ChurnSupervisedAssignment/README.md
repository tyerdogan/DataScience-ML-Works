# Telecom Customer Churn Prediction

Predicting customer churn using supervised machine learning on the Telco Customer Churn dataset.

## Dataset
- **Source:** IBM Telco Customer Churn Dataset
- **Size:** 7,032 customers with 19 features
- **Target:** Churn (Yes/No)

## What's Inside
- Data cleaning and exploratory analysis
- Feature engineering with proper encoding and scaling
- 8 classification models trained and compared
- Hyperparameter tuning using RandomizedSearchCV
- Confusion matrices and feature importance analysis

## Results
- **Best Model:** Gradient Boosting
- **ROC-AUC:** 0.8418
- **Accuracy:** 79.95%
